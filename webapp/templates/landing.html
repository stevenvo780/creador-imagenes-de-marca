Por qué Eikón

Tres promesas, una herramienta.

Diseñado para ser predecible, accesible y compartir el mismo sistema visual entre todas las marcas del ecosistema. Misma semilla, mismo render — en cualquier lugar.

$0 en tokens

Sin API de pago, sin GPU, sin sorpresas. Algoritmos deterministas que corren en cualquier runtime: navegador, Node, Python, Bun, lo que tengas.

Isotipos procedurales

Geometrías simples — círculo, hexágono, triángulo — que se recompone a partir de una semilla única. Cero assets binarios en el repo.

WCAG AA

Cada par de colores generado se valida contra contraste mínimo 4.5:1 antes de salir al mundo. Accesibilidad por defecto, no por accidente.

100% determinista

Misma semilla, mismo sistema. Reproducible hasta el último píxel, en cualquier plataforma. Lo que ves en local es lo que se sirve en producción.

Cómo funciona

Tres pasos: semilla, sistema, entrega.

Funciona como librería, CLI o API HTTP. Lo que elijas, el output es el mismo: un JSON firmado y reproducible que tu design system puede consumir directo.

  1. Paso

    Define la semilla

    Un slug humano — mouseion, prizma, elenxos. La misma palabra genera el mismo sistema en cualquier lado: docs, web, app.

  2. Paso

    Genera el sistema

    Paleta, isotipo, escala tipográfica y tokens de color/spacing/motion listos para enchufar en CSS, Swift, Kotlin o Figma Variables.

  3. Paso

    Exporta a tu stack

    JSON, SCSS, CSS variables, Swift, Kotlin — un solo comando, todas las plataformas. Versionado en Git, sin assets opacos.

API

Un endpoint. JSON reproducible.

Pasa una semilla, recibes el sistema completo. Cacheable, versionable, firmable. Lo mismo que consume tu web, lo consume tu CI. 

# GET /api/brand?seed=eikon
curl "https://eikon.humanizar.cloud/api/brand?seed=eikon" \
  | jq '.palette, .isotype, .tokens'
